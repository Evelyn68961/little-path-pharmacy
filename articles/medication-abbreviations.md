有天下午，一個讀大學的女生發 LINE 給我，傳了張藥袋的照片，上面寫著：「QID PO PRN」。

她問：「林藥師，這像不像在咒罵人？」

我笑了一下回她：「這意思其實是『一天四次、口服、需要時服用』。三個英文縮寫拼起來不友善，但拆開都不可怕。」

這篇文章，就是那次對話的延伸 — 把藥袋上最常見的縮寫整理一次，下次您拿到藥就能自己看懂。

## 為什麼藥師、醫師要用縮寫？

老實說，這些縮寫其實是 **拉丁文的簡寫**，沿用了好幾百年。在病人很多、處方手寫的年代，省時間。

現在大部分藥袋已經改成中文，但有些醫院的處方箋、藥袋的小角落、出院藥單，還是會看到。**了解一下，您就不會被嚇到。**

## 一、時間相關的縮寫

| 縮寫 | 拉丁原文 | 意思 |
|------|----------|------|
| QD | quaque die | 一天一次 |
| BID | bis in die | 一天兩次 |
| TID | ter in die | 一天三次 |
| QID | quater in die | 一天四次 |
| Q4H | quaque 4 hora | 每 4 小時一次 |
| Q6H | quaque 6 hora | 每 6 小時一次 |
| Q8H | quaque 8 hora | 每 8 小時一次 |

> 小訣竅：Q 開頭通常是「每幾小時」，B/T/Q（不接 H）通常是「一天幾次」。

## 二、給藥途徑

| 縮寫 | 意思 |
|------|------|
| PO | 口服（per os） |
| SL | 舌下含服 |
| IV | 靜脈注射 |
| IM | 肌肉注射 |
| SC / SQ | 皮下注射 |
| TOP | 外用塗抹 |
| OD / OS / OU | 右眼 / 左眼 / 雙眼 |

最後一組眼藥水的縮寫特別重要 — 點錯眼會很麻煩。

## 三、特殊指示

| 縮寫 | 意思 |
|------|------|
| AC | 飯前（ante cibum） |
| PC | 飯後（post cibum） |
| HS | 睡前（hora somni） |
| PRN | 需要時服用（pro re nata） |
| STAT | 立刻服用 |

**PRN 是最容易誤會的一個**：它表示「不舒服才吃」，不是固定每天吃。例如止痛藥標 PRN，意思是 *如果痛了再吃，不痛就不要吃*。

## 四、分量相關

| 縮寫 | 意思 |
|------|------|
| 1 # | 一顆 / 一錠 |
| 1/2 # | 半顆 |
| tab | 錠（tablet） |
| cap | 膠囊（capsule） |
| ml | 毫升 |
| gtt | 一滴（眼藥水常用） |

## 一張隨身對照卡

我們在櫃檯有印好一張「藥袋縮寫對照卡」，名片大小，可以放錢包。**免費，問藥師就拿得到。**

下次去拿藥，如果看到任何縮寫不懂，就直接問。沒有任何問題是「太小白癡不該問」 — 您搞清楚了再吃藥，才是對自己負責。

那位大學生現在已經會自己看藥袋了。她媽媽前陣子拿到新藥也不慌張了 — 因為女兒可以幫忙翻譯。

知識就是這樣，一個人懂了，全家都安心一點。
